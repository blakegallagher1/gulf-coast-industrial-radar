# Ordered Task Backlog

This backlog is ordered from highest leverage to lowest for the first-pass product: real estate investors and developers using only free/public data sources.

## Highest Leverage

1. Define the investor/developer wedge.
2. Pick the launch geography.
3. Create a free-source coverage matrix.
4. Define the core data model.
5. Build the evidence archive first.
6. Build source registry schema and seed verified public/free sources.
7. Backtest 10 known Gulf Coast projects.
8. Build LED FastLane/ITEP ingestion.
9. Build LDEQ EDMS/public notice ingestion.
10. Build USACE New Orleans public notice ingestion.
11. Build SEC EDGAR keyword/watchlist ingestion.
12. Build parcel and land-control ingestion for one high-value parish.
13. Build base infrastructure ingestion for USGS/BTS/HIFLD-compatible layers.
14. Implement structured AI extraction for industrial-project claims.
15. Build entity resolution and free business-entity ingestion.
16. Build the Quiet Land Assembly Detector.
17. Build infrastructure proximity scoring.
18. Build incentive-application monitors.
19. Build environmental-permit monitors.
20. Build FERC, utility, and power-load monitors.
21. Build SEC and public-company filing monitors.
22. Build port, board, and local-agenda monitors.
23. Add SAM.gov and USAspending as late-stage confirmation signals.
24. Create the project-formation score.
25. Create stage labels.
26. Build investor/developer action logic.
27. Build the map-first UI.
28. Add evidence drawer and signal timeline.
29. Build watchlists and alerts.
30. Build analyst review workflow.
31. Build explainable AI extraction.
32. Build quality and false-positive evals.
33. Run 5-project backtest before MVP acceptance.
34. Tune scoring thresholds.
35. Add weekly investor brief generation.
36. Only then add contractor-specific recommendations.

## Critical Path

Free-source coverage -> evidence archive -> parcel/entity/site ingestion -> signal extraction -> project-formation scoring -> investor action logic -> weekly brief -> feedback loop.
