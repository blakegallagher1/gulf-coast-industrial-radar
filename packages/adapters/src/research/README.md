# Gulf Coast Source Schema Research Artifacts

This directory contains Markdown files generated by the `source-schema-researcher` dev-time utility
(`packages/agents/src/source-schema-researcher.ts`). Each file documents the current public API /
HTML schema of one Gulf Coast data source that the adapters scrape.

## Purpose

These artifacts serve as a reference for:

- Adapter developers who need to know what fields are available.
- The `AssemblyValidator` agent, which uses them as context when checking public coverage.
- Future schema migrations when a source changes its layout.

## Regenerating

```bash
npx ts-node packages/agents/scripts/research-sources.ts
# or for a specific source:
npx ts-node packages/agents/scripts/research-sources.ts --sources led-fastlane,la-sos
```

> **Note:** Regeneration costs ~$0.03–0.05 per source (sonar-deep-research model).
> The 7-day DB cache prevents redundant calls within a week.
