## 1. URL Pattern

EBR assessor parcel search API:

```
https://gis.brla.gov/api/assessor/parcels
```

| Parameter | Type | Description |
|---|---|---|
| `owner` | string | Owner name (contains) |
| `parcel_id` | string | Parcel ID |
| `sale_date_from` | YYYY-MM-DD | Sale date start |
| `sale_date_to` | YYYY-MM-DD | Sale date end |
| `min_sale_price` | number | Minimum sale price |
| `page` | integer | Page (1-based) |
| `per_page` | integer | Max 100 |

```bash
curl 'https://gis.brla.gov/api/assessor/parcels?sale_date_from=2025-01-01&min_sale_price=1000000&page=1&per_page=50'
```
