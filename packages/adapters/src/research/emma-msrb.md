## 1. URL Pattern

EMMA document search API:

```
https://emma.msrb.org/api/v1/documents/search
```

| Parameter | Type | Description |
|---|---|---|
| `q` | string | Issuer name or CUSIP |
| `docType` | string | `CR` (continuing disclosure) \| `OS` (official statement) \| `TR` (trade report) |
| `stateCode` | string | Two-letter state (e.g. `LA`) |
| `dateFrom` | YYYY-MM-DD | |
| `dateTo` | YYYY-MM-DD | |
| `page` | integer | 1-based |
| `pageSize` | integer | Max 50 |

```bash
curl 'https://emma.msrb.org/api/v1/documents/search?q=industrial+revenue&stateCode=LA&docType=CR&dateFrom=2025-01-01&page=1&pageSize=50'
```
