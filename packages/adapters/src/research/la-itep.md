## 1. URL Pattern

The ITEP public database search:

```
https://itep.louisiana.gov/api/v1/contracts/search
```

| Parameter | Type | Description |
|---|---|---|
| `q` | string | Company name or contract number |
| `parish` | string | Parish filter |
| `status` | string | `active` \| `expired` \| `pending` |
| `page` | integer | Page number (1-based) |
| `pageSize` | integer | Items per page (max 200) |

```bash
curl 'https://itep.louisiana.gov/api/v1/contracts/search?parish=St.+James&status=active&page=1&pageSize=50'
```

## 3. Pagination

Offset: `page * pageSize`. Continue while `page * pageSize < count`.
