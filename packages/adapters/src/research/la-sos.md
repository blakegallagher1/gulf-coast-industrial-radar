# Louisiana Secretary of State Commercial Search — Schema Research

> **Generated by** `source-schema-researcher` · model: sonar-deep-research · cost: $0.0301

## 1. URL Pattern

```
https://www.sos.la.gov/BusinessServices/SearchForABusiness/Pages/default.aspx
```

The SOS site uses an ASP.NET WebForms POST for search. The underlying API endpoint (discoverable via devtools) is:

```
https://coraweb.sos.la.gov/commercialsearch/CommercialSearch.aspx
```

With form fields:

| Field | Value |
|---|---|
| `__EVENTTARGET` | `btnSearch` |
| `txtEntityName` | Company name |
| `rblStatus` | `Active` \| `Inactive` \| `All` |
| `rblSearchType` | `StartsWith` \| `Contains` |
| `ScriptManager1` | (ASP.NET ScriptManager postback value) |

```bash
curl -X POST 'https://coraweb.sos.la.gov/commercialsearch/CommercialSearch.aspx' \
  -d '__EVENTTARGET=btnSearch&txtEntityName=Venture+Global&rblStatus=Active&rblSearchType=Contains'
```

## 2. Response Format

HTML page with a `<table id="searchResultsTable">` containing:

| Column | Description |
|---|---|
| Entity Name | Registered company name |
| Entity Type | LLC \| Corporation \| Partnership |
| Status | Active \| Inactive |
| Date Formed | YYYY-MM-DD |
| Parish | Registered parish |

## 3. Pagination

ASP.NET ViewState-based pagination. POST with `__EVENTTARGET=lnkNextPage` and carry the `__VIEWSTATE` token.
