## 1. URL Pattern

EDMS document search API:

```
https://edms.deq.louisiana.gov/app/api/documents/search
```

| Parameter | Type | Description |
|---|---|---|
| `facilityId` | string | DEQ facility identifier |
| `docType` | string | `APC` (Air Permit to Construct) \| `APO` (Air Permit to Operate) \| `WDP` (Water Discharge Permit) |
| `fromDate` | YYYY-MM-DD | Start date |
| `toDate` | YYYY-MM-DD | End date |
| `page` | integer | Page number |
| `pageSize` | integer | Results per page (default 20, max 100) |

```bash
curl 'https://edms.deq.louisiana.gov/app/api/documents/search?docType=APC&fromDate=2025-01-01&toDate=2025-12-31&page=1&pageSize=50'
```
