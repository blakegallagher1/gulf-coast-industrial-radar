## 1. URL Pattern

The FastLane public project search is at:

```
https://fastlaneng.louisianaeconomicdevelopment.com/projects/search
```

Query parameters:

| Parameter | Type | Description |
|---|---|---|
| `keyword` | string | Free-text search (company name, project name) |
| `status` | string | `announced` \| `under_construction` \| `completed` |
| `parish` | string | Parish name (e.g. `Ascension`) |
| `page` | integer | 1-based page number |
| `per_page` | integer | Results per page (default 25, max 100) |

Example:
```bash
curl 'https://fastlaneng.louisianaeconomicdevelopment.com/projects/search?status=announced&parish=Ascension&page=1&per_page=25'
```

## 2. Response Format

Returns JSON with the following top-level shape:

```json
{
  "total": 142,
  "page": 1,
  "per_page": 25,
  "projects": [ /* array of ProjectRecord */ ]
}
```

### ProjectRecord fields

| Field | Type | Description |
|---|---|---|
| `id` | string (UUID) | Internal LED project identifier |
| `name` | string | Project name |
| `company` | string | Investing company |
| `parish` | string | Louisiana parish |
| `announced_date` | ISO 8601 | Date announced |
| `investment_usd` | number | Capital investment in USD |
| `jobs_created` | number | Estimated permanent jobs |
| `status` | string | Project lifecycle status |
| `naics_code` | string | NAICS classification |
| `description` | string | Public summary |

## 3. Pagination

Standard offset pagination: increment `page` while `page * per_page < total`.
