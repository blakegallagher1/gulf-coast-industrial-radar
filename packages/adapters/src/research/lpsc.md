## 1. URL Pattern

LPSC eFiling case search:

```
https://lpscefiling.lpsc.la.gov/WebAPI/api/Cases/Search
```

| Parameter | Type | Description |
|---|---|---|
| `caseType` | string | `U` (utility) \| `T` (telecom) \| `E` (electric) |
| `startDate` | MM/DD/YYYY | Filed on or after |
| `endDate` | MM/DD/YYYY | Filed on or before |
| `keyword` | string | Free text |
| `pageNumber` | integer | 1-based |
| `pageSize` | integer | Max 100 |

```bash
curl 'https://lpscefiling.lpsc.la.gov/WebAPI/api/Cases/Search?caseType=U&startDate=01/01/2025&endDate=12/31/2025&pageNumber=1&pageSize=50'
```

## 3. Pagination

Offset via `pageNumber`. Continue while `pageNumber * pageSize < TotalCount`.
