# SAM.gov Contract Opportunities — Schema Research

> **Generated by** `source-schema-researcher` · model: sonar-deep-research · cost: $0.0289

## 1. URL Pattern

```
https://api.sam.gov/opportunities/v2/search
```

| Parameter | Type | Description |
|---|---|---|
| `api_key` | string | SAM.gov API key (free registration) |
| `q` | string | Keywords |
| `postedFrom` | MM/DD/YYYY | Posted on or after |
| `postedTo` | MM/DD/YYYY | Posted on or before |
| `naicsCode` | string | NAICS code filter |
| `ptype` | string | `o` (solicitation) \| `p` (presolicitation) \| `k` (combined) |
| `limit` | integer | Results per page (max 1000) |
| `offset` | integer | Offset |
| `state` | string | Two-letter state code (e.g. `LA`, `TX`) |

```bash
curl 'https://api.sam.gov/opportunities/v2/search?api_key=YOUR_KEY&q=LNG+terminal&naicsCode=237120&state=LA&postedFrom=01/01/2025&limit=100&offset=0'
```

## 3. Pagination

Offset-based: `offset += limit` while `offset < totalRecords`.

## 4. Rate Limits

- Free API key: 100 req/hour.
- Public key: 10 req/hour (very restrictive; always use registered key).
