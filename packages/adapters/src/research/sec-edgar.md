## 1. URL Pattern

EDGAR full-text search API:

```
https://efts.sec.gov/LATEST/search-index?q=%22Gulf+Coast%22&dateRange=custom&startdt=2025-01-01&enddt=2025-12-31&forms=8-K,S-1
```

| Parameter | Type | Description |
|---|---|---|
| `q` | string | Search query (URL-encoded) |
| `dateRange` | string | `custom` |
| `startdt` | YYYY-MM-DD | Start date |
| `enddt` | YYYY-MM-DD | End date |
| `forms` | string | Comma-separated form types |
| `hits.hits._source` | string | Field projections |
| `from` | integer | Offset (0-based) |
| `size` | integer | Results per page (max 100) |

```bash
curl 'https://efts.sec.gov/LATEST/search-index?q=%22Gulf+Coast+LNG%22&forms=8-K&startdt=2025-01-01&enddt=2025-12-31&from=0&size=20'
```

## 3. Pagination

Offset-based: increment `from` by `size` while `from < hits.total.value`.
