## 1. URL Pattern

TCEQ Integrated Compliance and Reporting (ICRS) air permit search:

```
https://www15.tceq.texas.gov/crpub/index.cfm?fuseaction=iwr.air&action=search
```

Form POST fields:

| Field | Description |
|---|---|
| `crit_applicant` | Applicant company name |
| `crit_county` | Texas county name |
| `crit_status` | `PND` (pending) \| `ACT` (active) \| `ALL` |
| `crit_apptype` | `NSR` (new source review) \| `PSD` \| `MST` |
| `crit_date_from` | MM/DD/YYYY |
| `crit_date_to` | MM/DD/YYYY |

```bash
curl -X POST 'https://www15.tceq.texas.gov/crpub/index.cfm?fuseaction=iwr.air' \
  -d 'action=search&crit_county=Jefferson&crit_status=PND&crit_apptype=NSR&crit_date_from=01/01/2025&crit_date_to=12/31/2025'
```
